# The Mental Offload — legal pages

Static HTML for YahtzEats and BoopTime (Google Play, App Store, privacy forms).

## Google Sites (quick)

1. Create a new site at [sites.google.com](https://sites.google.com).
2. Add pages and **copy/paste** text from each HTML file (Sites does not import this zip directly).
3. Publish the site. Use your **Delete account** page URL in Play Console.

**Play Console “Delete account URL”:** publish `delete-account.html` content as its own page, e.g.  
`https://sites.google.com/view/your-site/delete-account`

## Other free hosting (full HTML zip)

- **GitHub Pages:** push `site/` folder, enable Pages on `main` / `/site`.
- **Cloudflare Pages / Netlify:** drag-and-drop the `mental-offload-site.zip` contents.

## Files

| File | Use |
|------|-----|
| `index.html` | Home |
| `privacy.html` | Privacy policy URL |
| `delete-account.html` | **Required** Play Store account deletion URL |
| `eula.html` | Terms / EULA |
| `styles.css` | Shared styling (keep in same folder when hosting) |
